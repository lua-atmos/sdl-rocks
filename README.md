# The "Rocks!" game

A spaceship shooter game for 2 simultaneous players.

# Run

Requires [lua-atmos](https://github.com/lua-atmos/atmos):

```
lua5.4 main.lua
```

# Instructions

- Hit the other ship.
- Avoid the rocks!
- Controls:
    - Left Ship: `WASD` to move, `Shift Left` to shoot.
    - Right Ship: Arrow keys to move, `Shift Right` to shoot.
